== Description ==

This plugin pulls product pricing, Description/Details, currency and order URL from the Faveo Agora invoicing web application.

== Installation ==

1. Upload the `agora-invoicing` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin by going to Settings -> Agora Invoicing.

== Frequently Asked Questions ==

= How do I set the API URL? =

Go to Settings -> Agora Invoicing and enter the API URL.
